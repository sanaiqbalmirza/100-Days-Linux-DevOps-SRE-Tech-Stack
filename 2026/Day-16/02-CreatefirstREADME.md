# Task 3 - Using ChatGPT to Improve Documentation

## Objective

Use ChatGPT to help improve and organize your markdown documentation.

---

## Step 1

Copy all the modified content from your:

```text
README.md
```

file.

---

## Step 2

Paste the content into ChatGPT.

---

## Step 3

Ask ChatGPT:

```text
Please create a professional single markdown format document for me.
```

---

## Step 4

Once ChatGPT generates the markdown document:

- Copy the generated markdown
- Paste it back into your README.md file
